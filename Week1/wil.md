#git

git은 버전 관리 및 협업을 위해 사용하는 오픈소스 소프트웨어다.
git으로 파일을 관리하기 위해서는 디렉토리에 git저장소를 만들고, git으로 관리할 대상을 등록하며, 파일을 로컬 저장소로 옯겨야 한다.
파일을 로컬저장소로 옮기려면 git push origin main을 사용하면 된다.
